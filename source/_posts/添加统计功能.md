## 全局配置

busuanzi_count的配置项下enable: true

## 站点UV配置

```
site_uv: true
site_uv_header: 本站访客数
site_uv_footer: 人次

```

## 站点PV配置

```
site_pv: true
site_pv_header: 本站总访问量
site_pv_footer: 次

```

## 单页面PV配置

```
page_pv: true
page_pv_header: 本文总阅读量
page_pv_footer: 次

```
